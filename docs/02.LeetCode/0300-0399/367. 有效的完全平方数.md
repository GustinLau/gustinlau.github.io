## 题目

给定一个 **正整数** `num` ，编写一个函数，如果 `num` 是一个完全平方数，则返回 `true` ，否则返回 `false` 。

## 示例

示例 1：

```text
输入：num = 16
输出：true
```

示例 2：

```text
输入：num = 14
输出：false
```

## 提示

- 1 <= num <= 2<sup>31</sup> - 1

**进阶**：**不要** 使用任何内置的库函数，如 `sqrt` 。

## 分析

二分查找

## 解答

```javascript
/**
 * @param {number} num
 * @return {boolean}
 */
var isPerfectSquare = function (num) {
  let l = 0,
    r = num
  while (l <= r) {
    const mid = Math.floor((r - l) / 2) + l
    const result = mid * mid
    if (num === result) {
      return true
    } else if (num > result) {
      l = mid + 1
    } else {
      r = mid - 1
    }
  }
  return false
}
```
