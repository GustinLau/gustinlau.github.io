## 题目

给定一个整数数组，判断是否存在重复元素。

如果存在一值在数组中出现**至少两次**，函数返回 `true` 。如果数组中每个元素都不相同，则返回 `false` 。

## 示例

示例 1:

```text
输入: [1,2,3,1]
输出: true
```

示例 2:

```text
输入: [1,2,3,4]
输出: false
```

示例 3:

```text
输入: [1,1,1,3,3,4,3,2,4,2]
输出: true
```

## 分析

可以利用集合特性，将元素去重然后对比长度

## 解答

```javascript
/**
 * @param {number[]} nums
 * @return {boolean}
 */
var containsDuplicate = function (nums) {
  return Array.from(new Set(nums)).length !== nums.length
}
```
